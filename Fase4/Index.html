Páginas web utilizando un framework CSS aplicando estrategias de posicionamiento para el SEO

… “una media query consiste en un tipo de medio y al menos una consulta que limita las hojas de estilo utilizando características del medio como ancho, alto y color.
Se entiende como un módulo CSS3 que permite adaptar la representación del contenido a características del dispositivo, lo que se conoce diseño líquido o responsive.” (Rincón, s. f.)
“Las media queries permiten a las páginas web utilizar diferentes declaraciones de estilo de la hoja de estilos en base al ancho de la pantalla del dispositivo en el que se visualiza el sitio web. En los diseños web responsive, una media query es una declaración CSS que se utiliza como parámetro para saber cuándo utilizar otra declaración de estilo en base a las dimensiones del dispositivo en donde se visualiza la página web. Hay dos maneras de utilizar una media query: utilizando una hoja de estilos externa o escribiendo directamente en la hoja de estilos.” (staffcreatica, 2016)
“Las media queries actúan como contenedor de las reglas y selectores a aplicar, por lo tanto, todo lo que engloba empieza con la apertura de una llave y termina con el cierre de la misma. En cuanto a la sintaxis, una media query siempre comienza con el indicador de @media, seguido de un mediatype.” (Media Queries en CSS ¿Cómo funcionan?, 2016)

De acuerdo con los autores consultados, los media queries son directrices que se le dan a la página web para que autoajuste los estilos que se encuentran en lenguaje css al tamaño de los dispositivos en los que se va a mostrar. Surge de la necesidad de tener un sitio web que responda principalmente al auge de los dispositivos móviles y con el fin de no tener dos páginas una para escritorio y otra para las aplicaciones móviles sino una sola que se auto adapte a la fácil usabilidad dentro de un pequeño equipo como el celular
